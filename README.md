# 📚 Sosyal Bilgiler — دليل دراسة تفاعلي (تركي ⇄ عربي)

صفحة دراسية تفاعلية ثنائية اللغة لمراجعة مادة الاجتماعيات (Sosyal Bilgiler) قبل الامتحان.
**التركية على اليسار، والعربية تقابلها على اليمين تماماً** مع محاذاة دقيقة وخط فاصل في المنتصف.

> Türkçe terimler **solda**, Arapça karşılıkları **tam karşısında sağda** — hassas hizalama ile.

## ✨ المميزات / Özellikler
- 🔑 تظليل الكلمات المفتاحية للامتحان في اللغتين.
- 🏰 خط زمني بصري للمعارك (باسينلار → ملاذكرد → ميريوكيفالون).
- 💻 بطاقات التطبيقات الرقمية (e-Nabız, e-Devlet, e-Okul, CİMER).
- 🧩 لعبة مطابقة الموارد بالأنشطة + 🎯 لعبة تصنيف (براءة الاختراع / حقوق النشر).
- ⚖️ سيناريو حقوق المواطنة، و✅ اختبار من 12 سؤالاً مع تصحيح فوري ونتيجة.
- 🌙 وضع ليلي، شريط تقدّم، وتصميم متجاوب مع الجوال.
